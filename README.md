# 📝 To-Do Web App

A clean, minimal and responsive **To-Do List Web Application** built using **HTML, CSS, and JavaScript**.  
This project helps users easily add and manage daily tasks in a simple interface.

## ✨ Features

- ➕ Add new tasks with title & description  
- 🗑️ Delete tasks instantly  
- 📱 Fully responsive design  
- ⚡ Fast and lightweight (no frameworks)  
- 🎯 Simple and user-friendly UI  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## 💡 How It Works

1. Enter task title and description  
2. Click **Save**  
3. Task appears in the table  
4. Click **X** to delete task
